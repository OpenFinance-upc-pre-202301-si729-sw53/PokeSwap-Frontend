How to Trade on Our Platform


Please note that all of these steps are performed automatically. You just need to learn the basic functionality and start trading!


  • Step 1: Select a Currency

    Currencies are a crucial element in starting your trading journey. Go to the Currencies tab and select the one that best suits your needs.


  • Step 2: Select a Token

    Tokens are the coins that you will be trading. In the first version, you have some bonus balance, so select the token that you like best and start trading!


  • Step 3: Make the Exchange

    As this is a demo version, you will be performing exchanges with the platform. Don't worry, in the next version, you will be able to trade with other people!
